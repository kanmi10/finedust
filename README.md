# 😷  오늘미세어때?

##   프로젝트 소개

>  한국환경공단에서 제공하는 대기오염정보 API를 활용한 전국 미세먼지 측정 서비스
<br>


## 프로젝트 목적
- API를 요청하고 데이터를 반환받아 가공하는 과정 학습
- 간단한 로그인 구현을 통한 인증&검증 학습

<br>
